# シンプル電卓

スタンドアロンのHTMLファイルだけで動作する電卓です。ブラウザで`index.html`を開くだけで、四則演算やパーセント計算ができます。

## 特徴
- **シンプルなレイアウト**: 4×4グリッドのキー配置と見やすい表示エリア。
- **ライト/ダーク対応**: `color-scheme`を指定し、OS設定に合わせて快適に利用できます。
- **入力バリデーション**: 安全な式評価と小数の入力制御で誤動作を防止。
- **パーセント変換**: 現在の数値を100で割った値を式に反映。
- **アクセシビリティ**: キーボード操作を完備し、`aria`属性で読み上げにも配慮。

## 使い方
1. リポジトリを取得後、`index.html`をブラウザで開きます。
2. 画面上のボタン、またはキーボードで数値や演算子を入力します。
3. `=`キーまたは`Enter`で計算結果を表示します。
4. `AC`で全クリア、`DEL`で一文字削除、`%`で現在の数値をパーセントに変換できます。

## キーボードショートカット
| 操作 | キー |
| --- | --- |
| 数字入力 | `0-9` |
| 小数点 | `.` |
| 演算子 | `+`, `-`, `*`, `/` |
| パーセント | `%` |
| 計算実行 | `Enter` または `=` |
| 一文字削除 | `Backspace` |
| 全クリア | `Escape` |

## ファイル構成
- `index.html`: すべてのHTML/CSS/JavaScriptが含まれた単一ファイル。
- `README.md`: このプロジェクトの概要と操作方法。
